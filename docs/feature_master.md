# ReFit — Feature Master

> **포지셔닝:** "많은 앱이 운동을 기록한다. ReFit은 AI로 운동 전을 준비하고, 운동 후를 회복시킨다."
> **원칙:** 운동 중 수동 기록 없음. 운동 데이터는 Apple Health / Health Connect 자동 동기화.
> **Last updated:** 2026-06-07

## 앱 플로우

```
로그인 / 회원가입
       │
       ▼
   온보딩 (최초 1회)
   루틴 → 수면 목표 → 신체 정보 → 헬스 연동 → 캐릭터
       │
       ▼
  ┌──────────────────────────────────────────┐
  │               메인 탭                     │
  │  홈  ·  운동  ·  커뮤니티  ·  프로필      │
  └──────────────────────────────────────────┘
            │
            ▼ (운동 시작 시)
  운동 전 체크 → 운동 중 → 운동 후 리포트
```

## 2. 온보딩 (최초 1회, 5단계)

| 단계 | 기능 | 백엔드 엔드포인트 | 상태 |
|------|------|------------------|------|
| 1/5 | 운동 루틴 설정 — 요일별 타깃 부위 / 휴식일 | `POST /users/me/onboard` | 🔌 |
| 2/5 | 수면 목표 — 취침·기상 시각 | `POST /users/me/onboard` | 🔌 |
| 3/5 | 신체 정보 — 키·몸무게·숙련도 | `POST /users/me/onboard` | 🔌 |
| 4/5 | 헬스 앱 연동 권한 요청 (선택) | 프론트 전용 (HealthKit / Health Connect) | 🔌 |
| 5/5 | 캐릭터 배정 & 전체 데이터 제출 | `POST /users/me/onboard` | 🔌 |

## 5. 운동 핵심 플로우

### 5-1. 운동 전 (Pre-Workout)

| 기능 | 백엔드 엔드포인트 | 상태 |
|------|------------------|------|
| 어젯밤 수면 시간 + 양호/부족 판정 | `GET /sleep/stats?days=1` | 🔌 |
| 오늘 날씨 | `GET /workouts/pre-message` (`weather_desc`) | 🔌 |
| 걸음수 (오늘) | 프론트 전용 — HealthKit / Health Connect | 🔌 |
| 안정 심박수 | 프론트 전용 — HealthKit / Health Connect | 🔌 |
| AI 운동 전 케어 팁 (Gemini) | `GET /workouts/pre-message` (`message`) | 🔌 |
| "운동 시작!" 버튼 | — | 🔌 |

### 5-2. 운동 중 (In-Workout)

| 기능 | 백엔드 엔드포인트 | 상태 |
|------|------------------|------|
| 세션 자동 시작 | `POST /workouts/sessions/start` | 🔌 |
| 오늘 타깃 부위 칩 표시 | 세션 시작 시 플랜 데이터 기반 | 🔌 |
| 경과 타이머 | 프론트 전용 | 🔌 |
| "운동 완료" → 헬스 데이터 읽기 → 세션 완료 제출 | `POST /workouts/sessions/{id}/complete` | 🔌 |
| ~~수동 세트/횟수/무게 입력~~ | — | 🗑 |

### 5-3. 운동 후 (Post-Workout)

| 기능 | 백엔드 엔드포인트 | 상태 |
|------|------------------|------|
| 세션 요약 카드 (시간·칼로리·XP·볼륨) | 세션 완료 응답 (`WorkoutSessionSummary`) | 🔌 |
| 레벨업 여부 표시 | 세션 완료 응답 (`level_up`) | 🔌 |
| AI 운동 후 회복 팁 (Gemini) | 세션 완료 응답 (`ai_feedback`) | 🔌 |
| "홈으로" 버튼 | — | 🔌 |

## 6. 프로필 (마이페이지)

| 기능 | 백엔드 엔드포인트 | 상태 |
|------|------------------|------|
| 캐릭터·닉네임·레벨·XP 표시 | `GET /users/me` | 🔌 |
| 프로필 수정 (닉네임) | `PATCH /users/me` | 🔌 |
| 인바디 최신 수치 표시 | `GET /users/me/inbody?limit=1` | 🔌 |
| 인바디 수치 직접 입력 | `POST /users/me/inbody` | 🔌 |
| 운동 숙련도 표시 | `GET /users/me` | 🔌 |
| 수면 목표 표시 / 수정 | `GET /users/me/sleep-goal`, `PUT /users/me/sleep-goal` | 🔌 |
| 운동 루틴 수정 | `PATCH /workouts/plans/{id}`, `DELETE /workouts/plans/{id}` | 🔌 |
| 알림 설정 | `GET /notifications/settings`, `PATCH /notifications/settings` | 🔌 |
| 로그아웃 | 프론트 전용 (토큰 삭제 + 스토어 초기화) | 🔌 |

## 8. 커뮤니티

> 친구 추가(닉네임 검색) 및 Co-op 배지는 백엔드 미구현 → 시연 범위 제외.
> 친구 목록·콕 찌르기·받은 응원은 시연 포함.

| 기능 | 백엔드 엔드포인트 | 상태 |
|------|------------------|------|
| 친구 목록 조회 | `GET /community/friends` | 🔌 |
| 친구에게 콕 찌르기 | `POST /community/pokes` | 🔌 |
| 받은 응원 목록 | `GET /community/pokes/received` | 🔌 |
| 친구 추가 (닉네임 검색) | ❌ 백엔드 미구현 | 🗑 |
| Co-op 배지 | ❌ 백엔드 미구현 | 🗑 |
