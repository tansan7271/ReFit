## 1. 전체 화면 흐름 순서도

```
앱 진입 (app/index.tsx)
   │
   ├─ 토큰 없음 / 온보딩 미완 ──────────────────────────────────────────────────────┐
   │                                                                                │
   │  [인증 그룹]                                                                   │
   │  ┌──────────────────────────────────────────────────────────────────────────┐ │
   │  │  로그인 ──► 회원가입                                                      │ │
   │  │  (login)    (register)                                                    │ │
   │  └──────────────────────────────────────────────────────────────────────────┘ │
   │                │ 로그인 성공 + 온보딩 미완료                                   │
   │                ▼                                                              │
   │  [온보딩 그룹 — 5단계 순차 진행]                                               │
   │  ┌──────────────────────────────────────────────────────────────────────────┐ │
   │  │  1/5 운동 루틴 ──► 2/5 수면 목표 ──► 3/5 신체 정보                       │ │
   │  │  (workout-routine)  (sleep-goal)    (physical-profile)                   │ │
   │  │        ──► 4/5 헬스 앱 연동 ──► 5/5 캐릭터 소개 & 완료                   │ │
   │  │            (health-connect)         (character-intro)                    │ │
   │  └──────────────────────────────────────────────────────────────────────────┘ │
   │                                  │ 온보딩 완료                                │
   │◄─────────────────────────────────┘◄───────────────────────────────────────────┘
   │
   │  로그인 성공 + 온보딩 완료
   ▼
[메인 탭 — 하단 네비게이션]
   ├─ 홈 (index)
   │    └──► 운동 전 체크하기 버튼
   │
   ├─ 운동 (workout)
   │    ├── 오늘의 플랜 조회
   │    ├── 최근 기록 요약
   │    └──► 운동 전 체크하기 버튼
   │              │
   │              ▼
   │         [운동 핵심 플로우]
   │         ┌─────────────────────────────────────────────┐
   │         │  운동 전 체크   ──►  운동 중   ──►  운동 후  │
   │         │  (pre-workout)      (session)    (post-workout) │
   │         └─────────────────────────────────────────────┘
   │                                                │
   │                                                └──► 홈으로 (replace)
   │
   ├─ 커뮤니티 (community)
   │    ├── 친구 탭
   │    ├── 받은 응원 탭
   │    └── Co-op 뱃지 탭
   │
   ├─ 뱃지 (badges)
   │
   └─ 프로필 (profile)
        ├── 인바디 모달 (InBodyModal)
        └──► 뱃지 컬렉션 (badges) 진입
             └── 로그아웃 ──► 로그인 화면
```
