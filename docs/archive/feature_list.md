# ReFit — Feature List

> **"Many apps track your workout. ReFit uses AI to guide you on how to prepare before, and how to recover after."**
>
> | Symbol | Meaning                                         |
> | ------ | ----------------------------------------------- |
> | ✅     | Done — backend API + frontend UI both connected |
> | 🔌     | Backend ready, frontend not yet wired           |
> | 🖼     | UI exists, backend not implemented              |
> | ❌     | Not implemented on either side                  |

---

## App Flow

```
Sign Up / Log In
       │
       ▼
   Onboarding (first time only)
   Routine → Sleep Goal → Body Profile → Health App → Character
       │
       ▼
  ┌─────────────────────────────────────────────┐
  │                 Main Tabs                   │
  │  Home  ·  Workout  ·  Community  ·  My Page │
  └─────────────────────────────────────────────┘
       │
       ▼ (when starting a workout)
  Pre-Workout Check → In-Workout → Post-Workout Report
```

## Features by Screen

### Onboarding

| Feature         | Description                                                                            | Status |
| --------------- | -------------------------------------------------------------------------------------- | ------ |
| Workout Routine | Select target muscle groups per day of the week; mark rest days                        | ✅     |
| Sleep Goal      | Set target bedtime and wake-up time; estimated sleep duration calculated automatically | ✅     |
| Body Profile    | Enter height, weight, and fitness level                                                | ✅     |
| Health App      | Connect Apple Health or Google Health Connect (optional)                               | ✅     |
| Character       | Receive a personal character that grows alongside you                                  | ✅     |

---

### Home

| Feature                 | Description                                                                        | Status |
| ----------------------- | ---------------------------------------------------------------------------------- | ------ |
| Character Status        | Character's appearance and mood change based on sleep quality and workout activity | ✅     |
| Character Level / XP    | Character grows as you work out more                                               | ✅     |
| Today's Workout Summary | Quick view of the workout plan scheduled for today                                 | ✅     |
| Condition Snapshot      | A one-line status message based on your sleep and today's weather                  | ✅     |

---

### Workout

| Feature         | Description                                                         | Status |
| --------------- | ------------------------------------------------------------------- | ------ |
| Today's Plan    | Automatically displays the workout list set for the current day     | ✅     |
| Session History | View past sessions with time, calories, volume, and XP earned       | ✅     |
| Weekly Stats    | Summary of completed sessions, total workout time, and total volume | ✅     |

### Pre-Workout Check

| Feature                 | Description                                                                                             | Status |
| ----------------------- | ------------------------------------------------------------------------------------------------------- | ------ |
| Smart Push Notification | Personalized pre-workout alert generated by Gemini AI based on sleep, weather, and fitness level        | 🔌     |
| Condition Dashboard     | Displays last night's sleep, step count, resting heart rate, and weather in one view                    | ✅     |
| AI Pre-Workout Guide    | Provides stretching tips, meal timing suggestions, and intensity adjustments based on today's condition | ✅     |

### In-Workout

### Post-Workout Report

| Feature              | Description                                                                                                           | Status |
| -------------------- | --------------------------------------------------------------------------------------------------------------------- | ------ |
